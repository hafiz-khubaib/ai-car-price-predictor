# 🚗 AI Car Resale Value Predictor

An intelligent Machine Learning web application that predicts the resale value of used cars with **86.62% accuracy** using a custom-tuned K-Nearest Neighbors (KNN) Regressor.

## 📊 Key Highlights
* **High Accuracy:** Achieved an $R^2$ Score of **86.62%** with a Mean Absolute Error (MAE) of only **0.99 Lakhs**.
* **Robust Preprocessing:** Handled extreme outliers (e.g., high mileage and luxury prices) using `RobustScaler` to ensure stable predictions.
* **Smart UI:** Created a sleek, mobile-friendly interface using Streamlit, featuring dynamic price range estimation.

## 🛠️ Tech Stack & Concepts Used
* **Language:** Python
* **ML Library:** Scikit-Learn (KNN Regressor, RobustScaler, Joblib)
* **EDA & Viz:** Pandas, NumPy, Seaborn, Matplotlib
* **Deployment:** Streamlit Cloud & GitHub
